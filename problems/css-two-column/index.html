CSS 两栏布局

1. Flex：左侧固定，右侧自适应

左侧 200px
右侧自适应
.layout {
  display: flex;
}

.left {
  flex: 0 0 200px;
}

.right {
  flex: 1;
  min-width: 0;
}

2. Grid：左侧固定，右侧自适应

左侧 200px
右侧自适应
.layout {
  display: grid;
  grid-template-columns: 200px minmax(0, 1fr);
}

3. Float：左侧固定，右侧自适应

左侧 200px
右侧自适应
.left {
  float: left;
  width: 200px;
}

.right {
  margin-left: 200px;
}

.layout::after {
  display: block;
  clear: both;
  content: "";
}

4. 响应式：窄屏变单列

@media (max-width: 768px) {
  .layout {
    display: block;
    grid-template-columns: 1fr;
  }

  .left,
  .right {
    width: auto;
    margin-left: 0;
  }
}
